stage5 task (Basic Calculator - My Choice)

This is a basic calculator designed by @archibongdaniel13 as a stage5 task during internship with hotels.ng

Note:

MC: clears the memory from the calculator

M+: Add numbers saved from previous calculation(s) to the current result

MS: Saves result into calculator memory

MR: Recall results added to M+

sqrt: Normal square root

1/x: Simply means 1 divided by any value say 'x' e. g 1/3 = 0.333333333 etc

x2: This is the same (the sqaure of any number) 0r 2 raised to power 2 which is equal to 2x2 = 4 etc
